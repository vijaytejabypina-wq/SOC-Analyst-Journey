# 🛡️ TryHackMe - Pre Security Path

This repository contains my notes, summaries, and hands-on learning from the TryHackMe Pre Security learning path.

The Pre Security path focuses on building a strong foundation in IT and Cyber Security concepts before moving into advanced security topics. It covers networking, web technologies, operating systems, and basic programming concepts. :contentReference[oaicite:0]{index=0}

---

## 📚 Learning Objectives

- Understand how computers and networks work.
- Learn the fundamentals of cyber security.
- Understand how websites and web applications function.
- Gain hands-on experience with Linux and Windows basics.
- Build foundational knowledge required for SOC, Blue Team, and Offensive Security paths. :contentReference[oaicite:1]{index=1}
